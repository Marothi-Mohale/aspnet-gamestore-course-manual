# ASP.NET GameStore Course Manual

A professional follow-along course manual for building a **GameStore Web API** with **C#**, **ASP.NET Core**, **Entity Framework Core**, **SQLite**, and **Swagger**.

This repository contains a structured PDF and DOCX guide designed to help learners build the project step by step in a practical way. The material is tailored to work with the **.NET 10 SDK**, so it can be followed without installing .NET 8 separately.

## What this repository includes

- A full follow-along course manual in **PDF**
- An editable **DOCX** version
- A step-by-step walkthrough of the GameStore API project
- Code snippets for easier implementation
- Notes adapted for **.NET 10 SDK**

## What you will learn

By following this manual, you will learn how to:

- set up an ASP.NET Core Web API project
- structure a clean backend project
- create entities and DTOs
- configure Entity Framework Core
- use SQLite with ASP.NET Core
- create Minimal API endpoints
- implement CRUD operations
- use Swagger for API testing
- apply migrations and database updates
- build a working GameStore API project from start to finish

## Tech stack

- C#
- ASP.NET Core
- .NET 10 SDK
- Entity Framework Core
- SQLite
- Swagger / OpenAPI
- Minimal APIs

## How to use this repository

1. Download or open the PDF manual.
2. Follow the steps in order.
3. Build the project as you go.
4. Use the code snippets to speed up implementation.
5. Test the API with Swagger.
6. Compare your result with the expected project structure in the guide.

## Notes

This manual was prepared as a study and follow-along resource based on the GameStore-style ASP.NET learning project. It is designed to make the build process easier to follow in a structured written format.

The guide was adapted for **.NET 10**, so some package versions and setup details were adjusted from older .NET course versions to match a newer SDK environment.
